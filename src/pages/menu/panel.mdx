# panel

这个样式设计来自于 [ui5 panel](https://sap.github.io/ui5-webcomponents/playground/?path=/docs/main-panel--docs)

<T style={{
    'border': '2px solid rgb(var(--black))',
    'border-radius': '16px',
    'overflow': 'hidden',
}}>
    <T tag="details" class="☘-panel">
        <summary>进入</summary>
        <T>到顶了</T>
    </T>
</T>

```pug
div(style={
    'border': '2px solid rgb(var(--black))',
    'border-radius': '16px',
    'overflow': 'hidden',
})
    details.☘-panel
        summary 进入
        div 到顶了
```


## `data-primary`

`details.☘-panel[data-primary]` 这个属性需要启用后才能继承主题和拥有某些局部样式

<T 
    tag="details" 
    class="☘-panel"
    data-primary="warning"
    style="color:rgb(var(--gold-8))"
>
    <summary>进入</summary>
    <T>到顶了</T>
</T>

```pug
details.☘-panel(
    data-primary="warning"
    style="color:rgb(var(--gold-8))"
)
    summary 进入
    div 到顶了
```

## custom style

全局变量 `--color-panel` 可以覆盖面板背景色

<T
    tag="details"
    class="☘-panel"
    style={{
        '--color-panel': 'var(--blue-1)'
    }}
    data-primary
    data-marker=">"
>
    <summary>
        <span>启动</span>
        <span>🥰</span>
    </summary>
    <T
        tag="p"
        class="☘-paragraph"
        data-size="2"
        style="margin:0"
    >😍😍😍😍😍😍</T>
</T>

marker图标由 `mask` 遮罩图片实现，可以自定义修改

```pug
details.☘-panel(
    data-marker=">"
    data-primary
    style={
    '--color-panel': 'var(--blue-1)'
})
    summary 
        span 启动
        span 🥰
    p.☘-paragraph(data-size=2 style="margin:0") 😍😍😍😍😍😍

style(lang=stylus).
    .☘-panel[data-marker="\>"] summary::before
        -webkit-mask url("@/icons/direction/outline/right.svg")
        mask @-webkit-mask
        background-color _rgb(var(--blue-6))
```
