## custom style

修改原生css变量 `--size-switch` 和 `--padding-switch` 以覆盖默认尺寸

<OxalisSwitch id="oxalis" />

```pug
span.☘-switch(
    style={
        ["--size-switch"]: "48px",
        ["--padding-switch"]: "4px",
        ["--primary-6"]: "var(--lime-3)",
    }
)
    input#oxalis.oxalis-switch(type="checkbox" checked)
    label(for="oxalis" title="oxalis")
        span.oxalis-switch-mover \☘

style(lang="stylus").
    .oxalis-switch>input:checked+label
        .oxalis-switch-mover
            transform rotate(360deg)
            overflow hidden 
```
