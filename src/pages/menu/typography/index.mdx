# typography

## font

`.☘-font-serif` 衬线字体

`.☘-font-mono` 等宽字体

`.☘-connot-select` 无法选中文字

和颜色一样，字体大小和行高有10个级别。

| lv | font-size | line-height |   alias   |
| -- | --------- | ----------- | --------- |
|  1 |      12px |        20px | base      |
|  2 |      14px |        22px | body / h6 |
|  3 |      16px |        24px |        h5 |
|  4 |      20px |        28px |        h4 |
|  5 |      24px |        32px |        h3 |
|  6 |      30px |        38px |        h2 |
|  7 |      38px |        46px |        h1 |
|  8 |      46px |        54px |           |
|  9 |      56px |        64px |           |
| 10 |      68px |        76px |           |

css变量 `--line-height-body` 和 `--font-size-body` 可以影响很多组件，

但标题其不受影响，需要自行选择`<h1/>`到`<h6/>`。

可以像下面这样写以改变全局css变量

```stylus
body
    --font-size-body var(--font-size-3)
    --line-height-body var(--line-height-3)
```

## code

主要用来拓展markdown

单行时 `code.☘-code` 同 `span.☘-tag` 样式没有区别。

但是塞进 `pre.☘-pre`里 可以用来显示多行代码

```pug
code.☘-code.☘-font-mono
    pre.☘-pre.☘-font-mono
        |    print("helloworld")
```
