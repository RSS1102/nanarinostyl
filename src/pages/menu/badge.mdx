## custom style

- 全局变量 `--badge-processing-weight` 调整 波浪大小
- 全局变量 `--badge-indent` 调整文字左内边距，原本是用 `text-indent` 属性实现，但在**Safari**中 `text-indent` 似乎有一部分标签不支持，且会导致波纹是**椭圆形**，这也可能是 `aspect-ratio 1 / 1` 兼容性影响。见下：

<T tag="span" class="☘-badge" data-processing data-primary="danger" style="
    --badge-indent: 0; 
    text-indent: 1.2em; 
    font-size: 20px; 
    --badge-processing-weight: 2px
">`debug`</T>

```pug
span.☘-badge(data-processing data-primary="danger" style={
    "--badge-indent": "0",
    "text-indent": "1.2em",
    "font-size": "20px",
    "--badge-processing-weight": "2px"
})
    span.☘-code.☘-font-mono debug
```

***

- 全局变量 `--badge-margin` 圆点的边距，可以用来反向调整圆点大小（你又小来他又大）

使用 `<svg/>` 需要设置容器或者其自身宽或高。

<T tag="span" class="☘-tag" data-primary="danger" style="--badge-indent:30px;--badge-margin:6px;border-style:dashed">
    <T tag="span" class="☘-badge" data-processing>
        <Icon name="general/outline/bug" alt="bug" width="30" height="30" style="vertical-align: middle"/>
    </T>
</T>

```pug
span.☘-tag(data-primary="danger" style={
    "--badge-indent": "30px",
    "--badge-margin": "6px"
    "border-style": "dashed"
})
    span.☘-badge(data-processing)
        svg(alt="bug" width="30" height="30" style="vertical-align: middle")
```
