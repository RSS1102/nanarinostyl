## custom style

`--badge-processing-weight` 调整 波浪大小

`--badge-padding` 反向调整 圆点大小

`--badge-indent` 调整 圆点和文字的距离

<T tag="span" class="☘-badge" data-processing data-primary="danger" style="font-size: 20px; --badge-processing-weight: 2px">`debug`</T>

<T tag="span" class="☘-badge" data-processing data-primary="danger" style="--badge-indent: 30px ;--badge-padding: 10px">
    <T tag="span" class="☘-tag" style="border-width:1pt">
        <Icon name="general/outline/bug" alt="bug" width="30" height="30" style="vertical-align: middle"/>
    </T>
</T>

```pug
span.☘-badge(data-processing data-primary="danger" style={
    "font-size": "20px",
    "--badge-processing-weight": "2px"
})
    span.☘-code.☘-font-mono debug

span.☘-badge(data-processing data-primary="danger" style={
    "--badge-indent": "30px",
    "--badge-padding": "10px"
})
    span.☘-tag(style="border-width:1pt")
        svg(alt="bug" width="30" height="30" style="vertical-align: middle")
```
