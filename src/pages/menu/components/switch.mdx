## custom style

修改原生css变量 `--switch-size` 以覆盖默认尺寸

<OxalisSwitch/>

```pug
span.☘-switch(
    style={
        "--switch-size": "48px",
        "--primary-6": "220 251 124",
    }
)
    input#oxalis-switch(type="checkbox" checked)
    label(for="oxalis-switch")
        span#oxalis-switch-mover \☘

style(lang="stylus").
    #oxalis-switch:checked+label #oxalis-switch-mover
        transform rotate(360deg)
        overflow hidden 
```
