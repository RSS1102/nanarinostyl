# custom style

  • --badge-processing-weight 调整 波浪大小
debug 
span.na-badge(data-processing data-primary="danger" style={
    "font-size": "20px",
    "--badge-processing-weight": "2px"
})
    span.na-code.na-font-mono debug
  • --badge-padding 反向调整 圆点大小
  • --badge-indent 调整 圆点和文字的距离

使用 <svg/> 需要设置容器或者其自身宽或高。 

span.na-badge(data-processing data-primary="danger" style={
    "--badge-indent": "30px",
    "--badge-padding": "10px"
})
