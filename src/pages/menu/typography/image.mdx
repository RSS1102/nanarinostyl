# image

![示例](./../../../assets/images/example.jpg)

这个组件的显示来自拓展后的markdown。但是加载失败。

```pug
p
    span.☘-image(style="display: inline-flex")
        img(src="/src/assets/images/example.jpg" alt="示例" onerror=function(){
            this.src="/src/assets/images/unknow.png"
        } ondragstart="return false")
            .☘-image-mask
                svg(width="24px" alt="eye" click=function(){
                    open("/src/assets/images/example.jpg")
                })
```

## `<img>`

以**容器**方式。

如果不提供 `width` 默认以块级占满。你也可以同上文一样设置 `display` 为行内。

在低版本浏览器中 `width` 必须提供

<T class='☘-image' style="width:300px">
    <img src={example} />
</T>

```pug
.☘-image(style="width:300px")
    img(src='/src/assets/images/example.jpg')
```

## `background-image`

以**背景**方式。

必须以某种方式指定 `height` 和 `background-image`

比如指定 `aspect-ratio`

<T class='☘-image' style={{
    height: '300px',
    'background-image': `url(${example})`
}}/>

```pug
.☘-image(style={
    height: '300px',
    'background-image': url('/src/assets/images/example.png')
})
```

## mask

背景图或容器都可以使用的遮罩 `.☘-image>.☘-image-mask`

<T class='☘-image' style={{
    width: '100px',
    height: '100px',
    'background-image': `url(${example})`,
    'border-radius': '50%'
}}>
    <T class='☘-image-mask'>
        <Icon name="interactive-button/outline/heart" alt="heart" width="24px"/>
    </T>
</T>


```pug
.☘-image(style={
    width: '100px',
    height: '100px',
    'background-image': url('/src/assets/images/example.jpg'),
    'border-radius': '50%'
})
    .☘-image-mask
        svg(width="24px" alt="eye")
```

### footer

适用于大尺寸图片（ 128像素以上 ）的底部遮罩 `.☘-image>.☘-image-footer`

<T class='☘-image' style="width:300px;height:300px;border-radius:var(--border-radius-lg)">
    <img src={example} />
    <T class='☘-image-footer'>
        <T class="☘-image-footer-content">
            <T class="☘-title" data-size='h5'> 标题标题标题标题标题标题标题标题标题标题标题 </T>
            <T class="☘-paragraph" data-ellipsis="l1"> 描述描述描述描述描述描述描述描述描述描述描述描述 </T>
        </T>
        <T class="☘-image-footer-action">
            <Icon class={escape('☘-link')} name="interactive-button/outline/eye" alt="eye" width="24px"/>
            <Icon class={escape('☘-link')} name="interactive-button/outline/download" alt="download" width="24px"/>
        </T>
    </T>
</T>

```pug
.☘-image(style={
    width: "300px",
    height: "300px",
    "border-radius": "var(--border-radius-lg)"
})
    img(src='/src/assets/images/example.jpg')
    .☘-image-footer
        .☘-image-footer-content
            .☘-title(data-size='h5') 标题标题标题标题标题标题标题标题标题标题标题

```
