# typography

## list

这个组件的显示来自拓展后的markdown。

### `ul>li.☘-list`

-  (R U R') U'  (R U R') 
-  (U R U' R’) 2次公式 
-  (U R U' R’) 3次公式 

```markdown
-  (R U R') U'  (R U R') 
-  (U R U' R’) 2次公式 
-  (U R U' R’) 3次公式 
```

### `ol>li.☘-list`

1.  R U R' U R U2 R' 
2.  (R' F R F') R U2' R' U (R U2' R') 

```markdown
1.  R U R' U R U2 R' 
2.  (R' F R F') R U2' R' U (R U2' R') 
```


## font

- `.☘-font-serif` 衬线字体
- `.☘-font-mono` 等宽字体
- `.☘-connot-select` 无法选中文字

和颜色一样，字体大小和行高有10个级别。

| lv | font-size | line-height |   alias   |
| -- | --------- | ----------- | --------- |
|  1 |      12px |        20px | base      |
|  2 |      14px |        22px | body / h6 |
|  3 |      16px |        24px |        h5 |
|  4 |      20px |        28px |        h4 |
|  5 |      24px |        32px |        h3 |
|  6 |      30px |        38px |        h2 |
|  7 |      38px |        46px |        h1 |
|  8 |      46px |        54px |           |
|  9 |      56px |        64px |           |
| 10 |      68px |        76px |           |

全局变量 `--line-height-body` 和 `--font-size-body` 可以影响很多组件，

可以像下面这样写以改变全局变量使所有标签默认大小变成3级（即 `<h5/>` 的大小）

```sass
body
    --font-size-body: var(--font-size-3)
    --line-height-body: var(--line-height-3)
```

但标题其不受影响，需要自行选择 `<h1/>` 到 `<h6/>` ，对应markdown里的 `#` 数量。



## code

主要用来拓展markdown。不包括语法高亮。

单行时 `code.☘-code` 同 `span.☘-tag` 样式没有区别。

但是塞进 `pre.☘-pre`里 可以用来显示多行代码

```pug
code.☘-code.☘-font-mono
    pre.☘-pre.☘-font-mono
        |    print("helloworld")
```
