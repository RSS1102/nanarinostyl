# form

## `:focus-visible`

```sass
&:focus-visible
    outline-color rgb(var(--black))
```

使用键盘获取焦点时应该强制实现这个样式。默认是黑白，事实上浏览器已经实现。

- `button` tab或点击一次后回车触发
    - <T tag="button" class="☘-button">按钮</T>
- `a` tab或点击一次后回车触发
    - <T tag="a" class="☘-link" href="#">链接</T>
- `input` tab或点击一次后回车触发 但是普通的文本输入框不会触发
    - <T tag="input" class="☘-input" title="input" placeholder='请输入' />
    - <Switch id="switch" checked style="vertical-align: middle" />
- `textarea` 无法触发
    - <T tag="textarea" class="☘-textarea ☘-font-mono" title="textarea" placeholder='请输入' style="vertical-align: middle" />

使用全局css变量 `--outline-color-focus` 可以覆盖这个颜色

`input[data-primary]` 和 `textarea[data-primary]` 预设了 `--outline-color-focus` 主题色覆盖黑白。
