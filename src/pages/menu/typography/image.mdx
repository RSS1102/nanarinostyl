# image

组件扩展后的markdown设置图片尺寸将变得容易，注意不要使用换行

<T class='☘-image' style="width:100px">![未知](/images/unknow.png)</T>

```markdown
<div class="☘-image" style="width:100px">![未知](/images/unknow.png)</div>
```


## `<img>`

以**容器**方式。如果不提供 `width` 默认块级占满。在低版本浏览器中 `width` 必须提供

<T class='☘-image' style="width:300px">
    <img src='/images/example.jpg' />
</T>

```pug
.☘-image(style="width:300px")
    img(src='/images/example.jpg')
```

## `background-image`

以**背景**方式。必须以某种方式指定 `height` 和 `background-image`

<T class='☘-image' style="height:300px;background-image:url(/images/example.jpg)"/>

```pug
.☘-image(style={
    height: '300px',
    'background-image': url('/images/example.png')
})
```

## custom style

<T class='☘-image' style="width:200px;height:200px;border-radius:100px;overflow:hidden;margin:auto">
    <img src='/images/unknow.png' />
</T>

```pug
.☘-image(style={
    width: "200px",
    height: "200px",
    overflow: "hidden",
    "border-radius": "100px"
    margin: "auto"
})
    img(src='/images/unknow.png')
```
