# typography

`.☘-font-serif` 衬线字体

`.☘-font-mono` 等宽字体

`.☘-connot-select` 无法选中文字

和颜色一样，字体大小和行高有10个级别。

## `<code>` 和 `<pre>`

主要用来拓展markdown

单行时 `code.☘-code` 同 `span.☘-tag` 样式没有区别。

但是塞进 `pre.☘-pre`里 可以用来显示多行代码

```pug
code.☘-code.☘-font-mono
    pre.☘-pre.☘-font-mono
        |    print("helloworld")
```
