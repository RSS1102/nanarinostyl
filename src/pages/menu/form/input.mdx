# input

适用的类型：

- **`[type=text]`**
- `[type=search]`
- `[type=number]`
- `[type=password]`
- `[type=email]`
- `[type=tel]`
- `[type=url]`
- `[type=date]`
- `[type=time]`
- `[type=month]`
- `[type=week]`
- `[type=datetime-local]`
- `[type=datetime]`

<T tag="input" type="datetime-local" class="☘-input ☘-font-mono" title="更改时间" value='2023-09-08T14:48'/>

```pug
input.☘-input.☘-font-mono(type="datetime-local" title='更改时间' value='2023-09-08T14:48')
```

## `.sm` & `.lg`

不设置宽高时默认是21.5个等宽半角字符宽

`--padding-input` 设置内边距

有下面两个预设尺寸：

- `input.☘-input.sm`
    <br/><T tag="input" class="☘-input sm" title="helloworld" placeholder="更小" />
- `input.☘-input.lg`
    <br/><T tag="input" class="☘-input lg" title="helloworld" placeholder="更大" />


## affix

### `data-with-prefix`

这里使用`type="search"` 在**Safari**上会有额外的放大镜图标

<T class="☘-input-wrapper">
    <Icon class={escape('☘-input-prefix')} name="interactive-button/outline/search" alt="search" width="16px"/>
    <T tag="input" class="☘-input" data-with-prefix title="with-prefix" type="search"/>
</T>
```pug
.☘-input-wrapper
    svg.☘-input-prefix(alt="search" width="16px")
    input.☘-input(title='with-prefix' data-with-prefix type="search")
```

### `data-with-suffix`

<T class="☘-input-wrapper">
    <T tag="input" class="☘-input" data-with-suffix title="with-suffix" />
    <Icon class={escape('☘-input-suffix')} name="tips/outline/close" alt="close" width="16px"/>
</T>
```pug
.☘-input-wrapper
    input.☘-input(title='with-suffix' data-with-suffix)
    svg.☘-input-suffix(alt="close" width="16px")
```

上述属性和 `button` 一起使用可以增强语义，同时可以被键盘控制焦点。

当然 也可以是：
- `input[type=button]`
- `input[type=reset]`
- `input[type=submit]`

### `data-primary`

`☘-input-wrapper[data-primary]` 这个属性需要启用 启用后才能继承主题

<T class="☘-input-wrapper" data-primary="danger" style={{
    'border-style': 'solid'
}}>
    <T tag="input" class="☘-input" title="in-suffix" data-with-suffix />
    <T tag="button" class="☘-input-suffix" type="submit" title="search">
        <Icon name="interactive-button/outline/search" alt="search" width="18px"/>
    </T>
</T>
```pug
.☘-input-wrapper(data-primary="danger" style={
    'border-style': 'solid'
})
    input.☘-input(title='out-suffix')
    button.☘-input-suffix(type="submit" title="search")
        svg(alt="search" width="18px")
```
