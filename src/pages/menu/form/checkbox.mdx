# checkbox

<Checkbox id="hello">hello</Checkbox>
<br />
<Checkbox id="world" checked style="--checkbox-indent: 2em">world</Checkbox>

- 全局变量 `--checkbox-indent` 改变label文字左内边距，原本是用 `text-indent` 属性实现，，但在**Safari**中不生效，遂改用 `padding-left`

```pug
span.☘-checkbox
    input#hello.☘-input(type="checkbox")
    label(for="hello" title="hello") hello
br
span.☘-checkbox(style="--checkbox-indent: 2em")
    input#hello.☘-input(type="checkbox")
    label(for="hello" title="world") world
```

## `disabled`

<Checkbox id="HTTP" disabled>HTTP</Checkbox>
<Checkbox id="HTTPS" checked disabled dataList={{ primary : "success" }}>HTTPS</Checkbox>

```pug
span.☘-checkbox
    input#HTTP.☘-input(type="checkbox" disabled)
    label(for="HTTP" title="HTTP") HTTP
span.☘-checkbox(data-primary="success")
    input#HTTPS.☘-input(type="checkbox" checked disabled)
    label(for="HTTPS" title="HTTPS") HTTPS
```

## `data-indeterminate`

<Checkbox id="Superposition" dataList={{ indeterminate : true }} onchange="
    this.parentElement.removeAttribute('data-indeterminate')
">叠加态</Checkbox>

```pug
span.☘-checkbox(data-indeterminate)
    input#Superposition.☘-input(type="checkbox" onchange=function(){
        this.parentElement.removeAttribute('data-indeterminate')
    })
    label(for="Superposition" title="Superposition") 叠加态
```
