# font

  • .na-font-serif 衬线字体
  • .na-font-mono 等宽字体
  • .na-connot-select 无法选中文字

和颜色一样，字体大小和行高有10个级别。

css变量 --line-height-body 和 --font-size-body 可以影响很多组件，

可以像下面这样写以改变全局css变量使所有标签默认大小变成3级（即 <h5/> 的大小） 

body
    --font-size-body var(--font-size-3)
    --line-height-body var(--line-height-3)

但标题其不受影响，需要自行选择 <h1/> 到 <h6/> ，对应markdown里的 # 数量。

# code

主要用来拓展markdown。不包括语法高亮。

单行时 code.na-code 同 span.na-tag 样式没有区别。

但是塞进 pre.na-pre里 可以用来显示多行代码 

code.na-code.na-font-mono
    pre.na-pre.na-font-mono
        |    print("helloworld")
