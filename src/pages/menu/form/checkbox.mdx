# checkbox

<Checkbox id="hello">hello</Checkbox>
<br />
<Checkbox id="world" checked style="--checkbox-indent: 2em">world</Checkbox>

- 全局变量 `--checkbox-indent` 改变label文字左内边距，原本是用 `text-indent` 属性实现。但在Safari浏览器中不生效，遂改用 `padding-left` 来实现

```pug
span.☘-checkbox
    input#hello.☘-input(type="checkbox")
    label(for="hello" title="hello") hello
br
span.☘-checkbox(style="--checkbox-indent: 2em")
    input#hello.☘-input(type="checkbox")
    label(for="hello" title="world") world
```

## `data-indeterminate`

<Checkbox id="Superposition" dataset={{ indeterminate : true }} onchange="
    this.parentElement.removeAttribute('data-indeterminate')
">叠加态</Checkbox>

```pug
span.☘-checkbox(data-indeterminate)
    input#Superposition.☘-input(type="checkbox" onchange=function(){
        this.parentElement.removeAttribute('data-indeterminate')
    })
    label(for="Superposition" title="Superposition") 叠加态
```

## custom style

- `--color-checkbox` 全局变量覆盖颜色
- `--color-checkbox-disabled` 禁用状态全局变量覆盖颜色

<Checkbox
    id="BlueSuperposition"
    disabled
    dataset={{ indeterminate : true }}
    style={{
        '--color-checkbox': 'var(--blue-6)',
        '--color-checkbox-disabled': 'var(--blue-2)'
}}>饿死了</Checkbox>

```pug
span.☘-checkbox(data-indeterminate style={
    '--color-checkbox': 'var(--blue-6)',
    '--color-checkbox-disabled': 'var(--blue-2)'
})
    input#BlueSuperposition.☘-input(type="checkbox" disabled)
    label(for="BlueSuperposition" title="BlueSuperposition") 饿死了
```
